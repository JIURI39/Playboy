## 功能 & 开发进度

- [x] 主题颜色自定义, 完全深色模式支持
- [x] 播放本地和媒体文件.
- [x] 设置为系统打开方式
- [x] 解析BV链接.
- [x] 便携版
- [x] 文件下载功能.
- [ ] 编辑媒体文件.
- [ ] 多语言支持
- [ ] 移动端 (android) 支持
- [ ] 跨桌面端支持 (Windows, Linux, macOS)
