## 功能

- [x] 打开本地或网络文件.
- [x] 解析部分网站链接.
- [x] 便携版
- [x] 下载网络视频.
- [ ] 编辑媒体文件.
- [ ] 多语言支持
- [ ] 移动端 (android) 支持
